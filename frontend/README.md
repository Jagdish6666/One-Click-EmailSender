Next.js app (UI only). Calls backend API for auth and data.

## Setup

1. `npm install`
2. Copy `.env.example` to `.env` and set NEXT_PUBLIC_API_URL (e.g. http://localhost:3001)

## Run

```bash
npm run dev
```

Runs on http://localhost:3000.

Make sure the backend is running on the URL set in NEXT_PUBLIC_API_URL.
